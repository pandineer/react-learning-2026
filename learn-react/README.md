#REACTを学ぶ

## UIの記述

### 初めてのコンポーネント

* Reactでは、マークアップとCSSとJavaScriptを、独自の”コンポーネント”と呼ばれる、アプリのための再利用可能なUI要素にまとめることができる
  * Reactコンポーネントとは、マークアップを添えることができるJavaScript関数である
  * Reactコンポーネントは普通のJavaScript関数であるが、名前を大文字から始めないと動作しない
  * コンポーネント内のタグの記述はHTMLのように書かれているが、実際にはJSX構文と呼ばれるもので、これによりマークアップをJavaScript内に埋め込むことができる
* 最終的にブラウザに見えるのはHTMLのタグで、Reactコンポーネントは見えない
* コンポーネントがほかのコンポーネントをレンダーすることができるが、とても遅くバグの原因になるため、コンポーネントの定義をネストさせてはいけない
  * すべてのコンポーネントをトップレベルで定義する
  * 子コンポーネントが親コンポーネントの情報を必要とする場合は、コンポーネント定義をネストさせるのではなくpropsを通じて渡す
* まとめ
  * Reactによりアプリのための再利用可能な部品であるコンポーネントを作成できる
  * ReactアプリではUIのあらゆる部品はコンポーネントである
  * Reactのコンポーネントとは普通のJavaScript関数だが、以下の点が異なる
    * 名前は常に大文字で始まる
    * JSXマークアップをreturnする
